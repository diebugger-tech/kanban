# Contributing to SurKAi

First off, thanks for taking the time to contribute! 🎉

## How to Contribute
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. Ensure the test suite passes (if applicable).
4. Update documentation as needed.

## PR Guidelines
- Follow the "Single-File Rule" where possible.
- Keep commits small and descriptive.
- Include screenshots or GIFs in your Pull Request if you modify the UI.

## Code Style
- **Aesthetic**: Maintain the "Terminal Dark Theme". Colors should stick to the existing palette (`#0a0a0b` background, `#00ffaa`, `#00aaff`, `#ffaa00` accents).
- **Font**: Always use `JetBrains Mono` or similar monospace fonts for UI elements.
- **React**: Functional components, Hooks, and clean prop drilling (or Context if it scales).

We look forward to your contributions!
